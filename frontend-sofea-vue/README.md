# Desafio Técnico - AZ Informática | Frontend - SOFEA Vue

Este projeto faz parte do desafio técnico proposto pela empresa AZ Informática como parte de seu processo seletivo para contratação.


## Getting Started

As instruções contidas nesse arquivo tem a finalidade de clarificar os procedimentos necessários para rodar a SOFEA (Service Oriented Frontend Application) desenvolvida em VueJS como frontend para o desafio.


### Prerequisites

Este projeto depende do NPM para rodar, que, por sua vez, é distribuído através do NodeJS. Mais informações a respeito da instalação do NodeJS e NPM para cada um dos sistemas operacionais mais modernos podem ser encontradas abaixo:

* [Node](https://nodejs.org/en/)


### Installing

O projeto em si não requer instalação. As únicas instalações necessárias são as mencionadas acima. Para clonar  o projeto para sua máquina, utilize um dos comandos a seguir em um diretório de escolha:

Com HTTPS

```
git clone https://github.com/DavidGuimaraes/desafio.git
```

Ou usando SSH

```
git clone git@github.com:DavidGuimaraes/desafio.git
```


## Deployment

Para rodar a aplicação, basta executar o comando abaixo dentro do diretório do frontend:

```
npm run serve
```
